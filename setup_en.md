# Streets of Rogue Setup Guide

## Required Software

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest).
- [BepinEx] (https://github.com/BepInEx/BepInEx/releases).
- [The SoR Archipelago Mod](https://github.com/Jmarcs27/StreetsOfRogueArchipelago/releases).

## BepinEx Setup

- Download the latest version of BepinEx. If the latest version is not working, for some reason, I've confirmed it to work on version 5.4.23.5.
- Open the zip file and put all contents in your streets of rogue directory. You can access this directory by right clicking on streets of rogue in the steam library search bar, click manage, then click browse local files.
- Run Streets of Rogue. Wait for the title screen to pop up and then quit the game.
- Return to the Streets of Rogue directory. Under the BepinEx folder, there should now be a plugins folder. 
- Place SoRArchipelago.zip in that plugins folder. Unzip SoRArchipelago.zip.
- Launch Streets of Rogue. If a connection window pops up in the title screen, the mod has been installed correctly and you're good to go!

## Additional Info
- To switch to a new save file in Streets of Rogue: Click on settings, then Gameplay, and scroll down to Change Save Slot. This will then put you in the tutorial. Make sure to connect to archipelago beforehand. Do not skip the tutorial! Sometimes the tutorial will not be completed as a check if you skip it. This should be fixed in the next version or three.
- Check if your starting characters line up with what you selected in the yaml. If not, you may have not connected to the archipelago properly or there is a bug. Check the troubleshooting section, and if that doesn't help, feel free to ping me in SoR thread on the archipelago discord.
- Use a fresh BepinEx zip from the BepinEx github page. Some other mods will provide custom BepinEx files that will not work with this mod. My brother had this issue, so learn from his mistake.
- Some of the names in the yaml may seem strange. I will fix this with aliasing in the future. 

## New player Info
- Don't get too frustrated if you die a lot. This is not an easy game. With experience, it can be made to be easy, but that does NOT make it an easy game, even for veterans. I played the game for over a year before realizing you can cook fud... and I just learned toilets have a use. Just take it slow and enjoy all the goofy interactions.
- If you want an easier character to start with, I recommend cop. Pick that with a few random characters characters and you should be able to have a good time.

## Troubleshoot
- Negative nuggets: You either unlocked something with nuggets obtained from outside the archipelago or you are playing on the wrong save file.
- No characters available to select when starting a run: You either haven't connected to the archipelago or connected after entering homebase. Leave homebase, connect to the archipelago if you haven't already, then re-enter homebase. The issue should be fixed.
- Tutorial achievement not marked as completed: If you skip the tutorial, it most likely won't mark itself as completed. Please mark as checked manually if you skipped.
- Big quests from other characters randomly completing: This should be fixed. If it is not, please let me know.
- Connection details not saving between play sessions: This is a feature that hasn't been implemented yet. Sorry, I know it's annoying.
